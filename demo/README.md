# TongTu Web Demo

A runnable chat interface for [TongTu](https://github.com/jesse-tzx/tongtu-china-travel) — the multilingual China travel assistant.

## Setup

1. Install the `flyai` CLI globally: `npm i -g @fly-ai/flyai-cli`
2. Copy `.env.local.example` to `.env.local` and add your Anthropic API key
3. Run `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Try These Queries

- **English**: "How do I pay in China?" / "I just landed at Pudong Airport, cheapest way to the Bund?"
- **Korean**: "항저우 서호 근처 호텔 추천해주세요" / "7월 25일에 서울에서 상하이 가는 비행기 찾아주세요"
- **Japanese**: "杭州の西湖近くで評判の良いホテルを教えてください"
- **Mixed**: "I want to visit Hangzhou next weekend. Find me flights from Seoul, hotels near West Lake, and tell me how to pay in China."
